离开蒙哥马利前的最后一个夜晚，阿欢在旅馆的房间里打开了系统面板。

**【历史进程面板】**
- 当前时间：1861年2月
- 距战争爆发：约2个月
- 南方邦联状态：已成立，宪法已颁布
- 北方联邦状态：林肯即将就职（3月4日）
- 战争概率：95%（萨姆特堡事件为触发点）
- 阿欢个人影响力：邦联层面 - 低；佐治亚州层面 - 中

**【资产状态】**
- 总资产估值：$620,000（含不动产）
- 可移动资产：$180,000
- 黄金储备：$35,000
- 武器装备：200杆步枪，弹药15,000发

阿欢关闭面板，在日记中写下了一段话：

```
// 1861.02.10 - 蒙哥马利最后一夜
// 邦联已诞生，宪法已颁布
// 它从出生起就带着致命的矛盾：
// 想要建国的权力，却不愿给政府必要的工具
// 想要赢得战争，却不愿改变奴隶制的低效
// 想要国际承认，却用棉花要挟而非外交

// 我无法改变这一切
// 但我可以：
// 1. 利用"工业特许状"扩建兵工厂
// 2. 利用"紧急接管"条款确保铁路运营
// 3. 在佐治亚层面推动实际的备战工作
// 4. 保护橡树庄园的每一个人

// 两个月。
// 两个月后，萨姆特堡的炮声将响起。
// 从那一刻起，一切都将不同。
```

他合上日记本，吹灭了蜡烛。

窗外，蒙哥马利的夜空寂静无声。远处的阿拉巴马河在月光下闪烁着银色的光芒，像一条通往未知未来的道路。

两个月。

他需要在这两个月里完成所有的准备工作。
