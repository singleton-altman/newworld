# 第四节

会议结束后，阿欢在蒙哥马利的街道上漫步。

二月初的阿拉巴马已经有了春天的气息。空气温暖而湿润，路边的山茱萸开始冒出嫩绿的芽苞。远处的阿拉巴马河在夕阳下泛着金色的光芒，几艘蒸汽船在河面上缓缓行驶，烟囱里冒出的白烟在微风中散开。

他的心情是复杂的。

一方面，他知道自己已经尽力了。在蒙哥马利的制宪会议上，他可能是唯一一个公开提出渐进改革和工业化方案的人。他的话虽然被否决了，但被记录了——在未来的历史书中，人们会看到，在邦联诞生的那一刻，有人曾经试图给它指明一条不同的道路。

另一方面，他感到一种深深的无力感。历史的惯性是巨大的——一个建立在奴隶制度上的社会，不会因为一个人的演讲就改变方向。保守派的力量太过强大，改革的声音太过微弱。

"赵先生。"

一个声音从背后传来。阿欢转身，看到一个五十多岁的男人站在他身后。那人身材瘦削，穿着一件简朴的灰色西装，面容严肃但不失温和。阿欢认出了他——杰斐逊·戴维斯，密西西比州的前参议员，即将被选为邦联的临时总统。

"戴维斯先生。"

"赵先生，我听了你在会议上的发言。"戴维斯走到他身边，两人并肩沿河而行。"很有见地。但也很……不合时宜。"

"我知道。"

"你知道吗，赵先生，"戴维斯的声音低沉而缓慢，"我私下里对你的某些观点是同意的。南方确实需要工业化。我们的经济基础确实过于单一。但是——"

他停下脚步，面对阿欢："但是政治不是做正确的事，而是在可能的范围内做尽可能多的事。在当前的南方，渐进废奴不在'可能的范围'内。你的提案虽然从经济角度有道理，但从政治角度是自杀。"

"那您认为正确的策略是什么？"

"团结。"戴维斯说了一个词，"在这个时刻，南方最需要的是团结。七个州刚刚脱离联邦，我们面对的是一个强大的敌人。如果我们内部还要在奴隶制问题上争论不休，我们在战争还没开始之前就已经输了。"

"可是，如果团结的代价是延续一个不公正的制度——"

"赵先生，"戴维斯打断他，声音依然温和但带着一种不容商量的坚定，"我不是来讨论哲学的。我是来建立一个国家的。在这个阶段，国家的需求优先于道德的需求。也许将来——战争结束之后——我们可以重新审视这个问题。但现在，我们需要每一杆枪、每一磅棉花、每一个能拿锄头的奴隶。"

阿欢沉默了。戴维斯的话让他想起了一个后世的词汇——"现实主义"。在政治中，现实主义往往是理想主义的敌人。而在这场即将到来的战争中，现实主义将主导一切。

"赵先生，"戴维斯在分手前说，"我听说你在佐治亚建立了一支不错的民兵队伍，还有一个颇具规模的纺织厂和铁路项目。这些——才是我们现在真正需要的。如果你愿意，我希望在邦联政府成立后，你能继续担任经济顾问的角色。我们需要能做事的人，而不仅仅是能说的人。"

"我愿意。"阿欢回答。不是因为认同戴维斯的所有观点，而是因为他知道：在即将到来的风暴中，他需要尽可能多的影响力——不是为了改变历史的走向，而是为了保护他所能保护的人。

回到旅馆后，阿欢给伊丽莎白写了一封长信。

*"亲爱的伊丽莎白：*

*蒙哥马利的天气比佐治亚暖和得多。路边的山茱萸已经发芽了，春天来得比往年早。*

*会议的情况如我所料——我的提案全部被否决。保守派的力量太强了。但我不后悔说出那些话。正如我对你说的，历史会记住这些声音。*

*戴维斯先生即将成为邦联的总统。他是一个有能力的人，但也是一个被时代束缚的人。他想建立一个国家，却不给这个国家建立成功所需的工具。这种矛盾，也许从一开始就注定了结局。*

*我在这里还有几天的工作。经济条款的讨论还在继续——虽然核心框架已经确定，但细节——关税税率、货币制度、贸易协定——仍然需要大量的谈判。我会尽力在这些细节中嵌入一些有利于我们长期战略的条款。*

*橡树庄园那边怎么样？解放之后的过渡是否顺利？杰克逊的铁匠铺有没有进展？孩子们识字课上得如何？这些'小事'，对我来说，比蒙哥马利的任何辩论都更有意义。*

*想念你。*

*欢*

*P.S. 告诉比利，我上次说的那个'雷管改良方案'——我想到了一个新的思路。等我回去详谈。"*

蒙哥马利的会议又持续了一个星期。

在这一个星期里，阿欢将精力集中在经济条款的细节上——他知道这些细节虽然不起眼，但对未来的战争经济至关重要。

他成功推动了两项关键条款的通过：

第一，邦联政府有权"在紧急情况下"接管铁路和通信设施。这条款看似普通，但在战争中，它将赋予邦联政府集中调度运输资源的权力——虽然历史上的邦联政府从未充分利用过这个权力。

第二，邦联政府可以颁发"工业特许状"，鼓励私人在南方建立军工厂和基础设施。这条款为阿欢自己的兵工厂计划提供了法律基础——他可以将橡树庄园附近的"金属加工厂"正式转为军工生产，而不会受到州政府的阻碍。

"小胜利，"他在日记中写道，"但在这个保守的堡垒里，小胜利已经是最大的收获了。"

二月九日，邦联宪法正式通过。杰斐逊·戴维斯被选为临时总统，亚历山大·斯蒂芬斯——佐治亚州的前联邦众议员——被选为临时副总统。

在就职典礼上，阿欢站在人群中，看着戴维斯在州议会大厦的台阶上宣誓就职。阳光照在新总统的脸上，他的表情庄严而自信。人群爆发出热烈的欢呼声——"南方万岁！""邦联万岁！""棉花王万岁！"

阿欢没有欢呼。他只是静静地站着，看着那个在阳光下宣誓的人，心中想着一个后世的判断：

杰斐逊·戴维斯是一个正直的、有能力的、有责任感的人。但他不是一个能创造奇迹的人。面对即将到来的战争——面对一个工业实力超过自己十倍、人口超过自己四倍的敌人——他需要的不仅仅是正直和能力。他需要的是奇迹。

而奇迹，从来不会降临在那些不愿意改变自己的人身上。

离开蒙哥马利前的最后一个夜晚，阿欢在旅馆的房间里打开了系统面板。

**【历史进程面板】**
- 当前时间：1861年2月
- 距战争爆发：约2个月
- 南方邦联状态：已成立，宪法已颁布
- 北方联邦状态：林肯即将就职（3月4日）
- 战争概率：95%（萨姆特堡事件为触发点）
- 阿欢个人影响力：邦联层面 - 低；佐治亚州层面 - 中

**【资产状态】**
- 总资产估值：$620,000（含不动产）
- 可移动资产：$180,000
- 黄金储备：$35,000
- 武器装备：200杆步枪，弹药15,000发

阿欢关闭面板，在日记中写下了一段话：

```
// 1861.02.10 - 蒙哥马利最后一夜
// 邦联已诞生，宪法已颁布
// 它从出生起就带着致命的矛盾：
// 想要建国的权力，却不愿给政府必要的工具
// 想要赢得战争，却不愿改变奴隶制的低效
// 想要国际承认，却用棉花要挟而非外交

// 我无法改变这一切
// 但我可以：
// 1. 利用"工业特许状"扩建兵工厂
// 2. 利用"紧急接管"条款确保铁路运营
// 3. 在佐治亚层面推动实际的备战工作
// 4. 保护橡树庄园的每一个人

// 两个月。
// 两个月后，萨姆特堡的炮声将响起。
// 从那一刻起，一切都将不同。
```

他合上日记本，吹灭了蜡烛。

窗外，蒙哥马利的夜空寂静无声。远处的阿拉巴马河在月光下闪烁着银色的光芒，像一条通往未知未来的道路。

两个月。

他需要在这两个月里完成所有的准备工作。
