# 第五节

那天夜里，火车在弗吉尼亚的丘陵地带缓慢行驶。

阿欢独自坐在指挥车厢的窗边，看着窗外飞驰而过的夜色。月光照亮了起伏的丘陵和蜿蜒的河流，银色的光芒洒在田野和树林上，给一切都镀上了一层梦幻般的色彩。

在车厢的另一端，几个军官已经睡着了——靠在椅子上，或者躺在铺在地板上的毯子上。他们的呼吸均匀而深沉，年轻的面孔在睡梦中变得柔和，没有了白天的紧张和故作镇定。

阿欢看着他们，心中涌起一种沉重的责任感。

这六百个人——加上工程连的四十三个人——他们的生命，在某种程度上，掌握在他的手中。他的每一个决定、每一个命令，都可能决定谁活着、谁死去。

在前世，他是一个程序员——一个在虚拟世界中构建系统的人。代码写错了可以重写，系统崩溃了可以重启。但在这里，在这个由血与火构成的真实世界中——

没有Ctrl+Z。

没有撤销。

没有重来。

他深吸一口气，打开日记本，写下了北上之路的最后一段记录：

```
// 1861.07.19 - 北上途中，距马纳萨斯约50英里
// 明天到达目的地
// 第一次牛奔河战役：预计7月21日

// 部队状态：
// 人员: 603+43 = 646人，全员到齐
// 士气: 中偏高（行军疲惫但精神状态良好）
// 装备: 已全面检查，状况良好
// 后勤: 独立补给链正常运作

// 心理准备：
// 作为一个穿越者，我知道这场战役的结果——
// 南军将获胜，北军将溃退。
// 但胜利不等于零伤亡。
// 历史上，南军在第一次牛奔河的伤亡约为1,900人。
// 我的任务是：让这646人中的尽可能多的人，
// 不在那1,900人之中。

// 个人状态：
// 我结婚了。
// 我有一个妻子在橡树庄园等我回去。
// 我有646个士兵在等着我带他们回家。
// 我有一个穿越者的知识——知道未来走向的优势。
// 
// 这些加在一起，就是我必须活下去的理由。
// 不是为了我自己——
// 而是为了那些信任我、依赖我、爱我的人。

// 明天见，弗吉尼亚。
// 明天见，牛奔河。
// 但愿这不是我们最后一次见面。
```

他合上日记本，闭上眼睛。

火车的轰隆声像一首催眠曲，在黑暗中持续回响。车轮碾过铁轨的节奏——咔嗒、咔嗒、咔嗒——如同一颗巨大的心脏在跳动，载着六百多个年轻的灵魂，向着命运的方向驶去。

窗外，弗吉尼亚的月亮挂在天空，又圆又亮。它的光洒在铁轨上，把两条平行的钢轨变成了两条银色的线，延伸到远方的黑暗中——延伸到那个无人能预知（除了阿欢）的未来。

阿欢在月光中沉入了睡眠。

在梦里，他看到了橡树庄园——看到了伊丽莎白站在主宅的走廊上，金色的头发在风中飘动。她微笑着向他招手。

而在她身后，在橡树的枝叶间，阳光穿透绿叶，洒下斑驳的光影。

那些光影，像是无数只温暖的手，在轻轻地抚摸这片土地。

七月二十日清晨，火车抵达了马纳萨斯。

阿欢走下火车，脚踩在弗吉尼亚的红土地上。空气中弥漫着泥土、汗水和火药的混合气味——这是战场特有的味道。

远处，传来隐约的炮声。

战争，近在咫尺。

他整了整军装，握紧了腰间的佩剑，然后转身面对他的六百个人。

"佐治亚第一步兵团——集合！"

六百个声音齐声回应，在弗吉尼亚的晨空中回荡。

北上之路结束了。

战争之路，从这里开始。
