# 第五节

圣诞节后的第一个星期，阿欢收到了两封重要的信。

第一封来自汉弗莱斯参议员。信中写道：

*"赵先生：你的'圣诞惊喜'已经在米利奇维尔引起了不小的轰动。州长本人对此表示了'关切'。一些议员在议会中要求调查橡树庄园是否违反了佐治亚州的奴隶法。我已经为你进行了辩护——主要论点是：这是私人庄园的内部事务，且你以'经济改革'的名义进行，不涉及政治目的。目前暂时压下了调查的动议，但我建议你低调行事，避免进一步刺激保守派。——S.H."*

第二封信来自一个意想不到的地方——弗吉尼亚的罗伯特·E·李。

*"赵先生：从佐治亚传来的消息让我颇感惊讶。你在圣诞夜做的事情，在很多人看来是鲁莽的。但在我看来，它是勇敢的——而且可能是有远见的。*

*我依然认为，奴隶制度的终结需要时间和渐进的过程。但我也相信，像你所做的这种'示范'——如果它能被更多的人看到和思考——也许能加速这个过程。*

*南方的未来取决于我们能否在保持尊严的同时做出改变。你在橡树庄园的尝试，也许是这种改变的一个微小的起点。*

*祝好。*
*R.E.Lee"*

阿欢读完李将军的信，将它仔细地收好。这封信——如果保存得当——将在未来的某个时刻成为一份珍贵的历史文献。

他把信放入书桌的暗格，然后站起身，走到窗前。

窗外，橡树庄园的冬日景色宁静而祥和。棉田休耕，田野空旷。远处的橡树林在灰色的天空下呈现一种肃穆的美感。在主宅前面的草坪上，圣诞树的蜡烛已经熄灭，但树枝上残留的彩色装饰物在冬日的阳光下仍然闪烁着微弱的光芒。

在草坪的一角，杰克逊——那个想开铁匠铺的年轻人——正跟着比利的一个助手学习基本的金属加工。他的动作生涩但认真，眼中带着一种阿欢从未在他脸上见过的光芒——那是"为自己而工作"的人才有的光芒。

在另一角，伊丽莎白在老汤姆的小屋里，给几个前奴隶的孩子上第一堂识字课。她用一本旧圣经作为教材，一个字母一个字母地教。孩子们的眼睛睁得大大的，充满了对未知世界的好奇和渴望。

在更远处的田野上，老汤姆独自走着，步伐比平时慢了许多。他不是在劳作，只是在散步——像一个自由人那样，在自己的土地上散步。

阿欢看着这一切，心中涌起一种深沉的满足感。

这不是终点。这只是一个开始。

战争很快就会来。炮火、鲜血、死亡——那些他知道即将到来的恐怖，没有任何力量可以阻止。但至少，在这个小小的角落里，在这个被橡树环绕的庄园中，八十多个人已经触摸到了自由的滋味。

哪怕只有一天。哪怕只有一瞬。

自由的味道，一旦被尝到，就永远不会被遗忘。

那天夜里，阿欢在日记本上写下了最后一段记录：

```
// 1860.12.26
// 橡树庄园解放完成
// 86人获得自由证书
// 78人选择留下作为雇佣工人
// 8人选择离开（已提供路费和推荐信）
// 白人雇员反应：70%接受/中立，20%不满，10%支持
// 自卫队未出现骚动
// 周边反应：预料中的强烈反对，但未出现暴力

// 核心成就：
// 1. 橡树庄园从奴隶制过渡到雇佣制
// 2. 建立了教育和培训体系
// 3. 为战后的"重建"模式提供了原型

// 下一步：
// 1. 应对政治反弹
// 2. 准备战争
// 3. 保护橡树庄园的每一个人——无论肤色

// 个人感悟：
// 穿越三年，我做了很多事——商业、工业、军事。
// 但今晚我觉得，这86份自由证书，
// 才是我来到这个世界后，最有价值的"输出"。
//
// 也许，这就是命运把我送到这里的原因。
// 不是为了改变历史的走向，
// 而是为了在历史的洪流中，
// 救起一些不该被淹没的人。
```

他合上日记本，吹灭了油灯。

窗外，1860年最后一天的星空格外清澈。繁星如同碎钻般散落在深蓝色的天幕上，银河横贯苍穹，像一条通往永恒的道路。

在这片即将被战火洗礼的土地上，在这个旧世界崩溃的前夜，阿欢闭上了眼睛。

他做了一个梦。梦里没有战争，没有炮火，没有鲜血。梦里只有橡树庄园的棉田——白色的棉絮在春天的微风中轻轻摇曳，像一片宁静的海洋。在棉田的尽头，老汤姆坐在摇椅上，对着夕阳微笑。

自由的味道，在空气中弥漫。
