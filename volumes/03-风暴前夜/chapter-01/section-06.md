# 第六节

春天，一个意想不到的访客来到了橡树庄园。

那天下午，阿欢正在庄园的"实验室"——一间由他改造的小房间，里面摆满了各种科学仪器、地图和书籍——里研究铁路建设的最新进展。门外传来管家玛格丽特夫人的声音："赵先生，有一位客人求见。他说他是从弗吉尼亚来的，名叫罗伯特。"

阿欢打开门，看到一个五十岁左右的男人站在走廊上。他身材挺拔，面容英俊而威严，花白的鬓角和修剪整齐的胡须赋予他一种古典罗马将军的气质。他穿着一件朴素的灰色旅行外套，没有任何军衔标志，但举手投足间带着一种与生俱来的军人气度。

"赵先生，"来者微微一笑，声音低沉而悦耳，"冒昧打扰。我叫罗伯特·李。途经佐治亚时，听几位朋友谈起了你的名字和你在这里做的一些有趣的事情。"

阿欢的心跳加速了半拍。罗伯特·E·李——美国内战史上最具传奇色彩的将军，此刻正站在他的走廊上，以一种旅行者的随和姿态向他微笑。

"李将军！"阿欢连忙将他迎进客厅，"不，应该说李上校——您目前是联邦军队工程兵部队的副主管？"

"你消息很灵通。"李在沙发上坐下，接过玛格丽特夫人端来的茶，"是的，我目前在华盛顿的工程兵总部任职。这次来南方是探亲——我的岳母住在拉文纳庄园，离这里不远。"

两人坐定后，李将军环顾客厅。橡树庄园的装饰风格在阿欢和伊丽莎白的共同打理下，兼具南方的温馨和东方的雅致——墙上挂着几幅中国水墨画的复制品（阿欢特意请人从旧金山的中国城购来），壁炉台上摆着一个精美的景泰蓝花瓶。

"很有意思的装饰，"李评价道，"我听说赵先生来自中国？"

"是的，将军。我十二岁时随家人移居美国。"这是阿欢为自己编造的"背景故事"。

"令人印象深刻。"李的目光变得深邃，"赵先生，我来拜访你，除了好奇之外，还有一个原因。最近我读了你在《萨凡纳信使报》上发表的一篇文章——《论南方经济自主与工业化的必要性》。你的观点很尖锐，也很……不合时宜。"

阿欢记得那篇文章。他刻意用了一个挑衅性的标题来吸引注意力，核心论点是：南方如果继续依赖棉花出口和奴隶劳动，将在经济上永远依附于北方和英国，最终在全球产业分工中沦为"永久殖民地"。

"将军觉得哪里不合时宜？"

"不是不合时宜，"李斟酌了一下措辞，"而是太尖锐了。你说的很多话，我私下里是同意的——南方需要工业化，需要经济多元化，需要减少对奴隶劳动的依赖。但你公开说出来，尤其是在当前的政治氛围下，只会让你树敌更多。"

阿欢认真地听着。他知道李将军是一个复杂的人物——作为弗吉尼亚贵族，他拥有奴隶，但从内心深处对奴隶制度持保留态度。他的妻子曾私下释放了多名家奴，而李本人在战争爆发后也曾考虑过渐进废奴的方案。

"将军，"阿欢直言道，"恕我冒昧——您认为奴隶制度能持续多久？"

李沉默了很长时间，长到阿欢以为他不会回答了。最终，他放下茶杯，缓缓说道：

"赵先生，奴隶制度是一个道德和政治上的困境。我个人的看法是——它终将消亡，但不是通过外力强加，而是通过南方人自身的觉醒。这是一个需要时间的过程，也许是二十年，也许是五十年。北方人要求立即废奴，却不了解南方的现实——我们的经济、社会结构、文化传统都与这个制度纠缠在一起。你不能一夜之间连根拔起一棵大树而不伤及土壤。"

"但如果时间不够呢？"阿欢追问道，"如果北方人等不了二十年，如果他们认为奴隶制度是对美国立国精神的根本背叛，如果矛盾激化到不可调和的地步——"

"那将是灾难，"李的声音变得沉重，"一场没有人能预料结局的灾难。我在墨西哥战争中亲眼目睹过战争的残酷。那些高喊'一战定乾坤'的年轻人，往往是最先倒下的。"

阿欢沉默了。他想告诉李将军——两年后，这场灾难就会降临。南卡会首先脱离联邦，然后其他南方州跟进，林肯会发布征兵令，弗吉尼亚会被迫选边站，而你，李将军，将被迫在联邦和故乡之间做出痛苦的选择。

但他不能。他只能默默地看着这位后世被无数人敬仰的军人，心中涌起一种复杂的感慨。

"赵先生，"李站起身来，"我很高兴认识你。你是一个有远见的人，但请记住我的忠告——在暴风雨来临之前，最明智的做法不是站在屋顶上大喊'风暴要来了'，而是悄悄地把房子加固。"

"我记住了，将军。"

李走到门口时，忽然回头："对了，你的那个棉花交易所——我在查尔斯顿的几个朋友很感兴趣。他们想邀请你去南卡做一次演讲。你愿意吗？"

阿欢心中一动。南卡罗来纳——未来第一个脱离联邦的州——向他发出了邀请。这是一个危险的机会，但也是一个了解南方核心政治圈层的窗口。

"当然愿意，将军。请替我转达谢意。"

送走李将军后，阿欢在书房里独坐到深夜。

他在日记本上写下了当天的思考——不是用中文，而是用一套只有他自己能看懂的"代码"。穿越以来，他养成了用编程思维记录重要信息的习惯：用if-then语句分析可能的历史走向，用变量名代表关键人物，用注释标注自己的推测和判断。

```
// 1859.03.15
// 关键事件：与R.E.Lee首次会面
// 印象：务实、理性、有道德底线但受制于时代局限
// 判断：战争爆发后，Lee将选择Virginia而非Union
// TODO：保持联系，未来可能成为关键人脉

if (战争爆发 == true) {
    南方军事优势 = 将领素质高 + 本土作战 + 骑兵强大;
    南方劣势 = 工业基础薄弱 + 人口劣势 + 海军为零 + 外交孤立;
    // 我的角色：弥补工业和后勤劣势
    // 关键：时间窗口只有2年
}
```

他合上日记本，走到窗前。橡树庄园的夜空繁星满天，银河如一条闪亮的丝带横贯苍穹。远处田野里传来蟋蟀的鸣叫和偶尔几声夜枭的低啼。

这片土地是美丽的。无论他来自哪里，无论他最初是如何不情愿地被抛入这个时代，他已经对这里产生了感情。橡树庄园的每一棵橡树、每一片棉田、每一条小溪，都承载着他两年多的汗水和心血。

而他知道，一场风暴正在酝酿。

历史的巨轮不会因为一个人的存在而改变方向——阿欢对此有清醒的认知。他不是小说里那种无所不能的穿越主角。他无法阻止南北战争的爆发，无法凭一己之力扭转整个南方的命运。但他或许可以做一些事情——保护他关心的人，为这片土地的未来埋下一些种子，在历史的洪流中为橡树庄园争取一线生机。

"至少，"他轻声对自己说，"我要让这里的人活下去。"
