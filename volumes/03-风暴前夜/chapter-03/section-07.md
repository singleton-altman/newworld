# 第六节

火车南下的旅途比来时更加漫长。阿欢靠在车厢的硬座上，闭着眼睛，脑海中不断回放着在弗吉尼亚看到的画面：铁链锁住的黑人、草率审判的法庭、狂热布道的牧师、以及李将军那双深邃而忧虑的眼睛。

他打开日记本，用那套只有他自己能看懂的"代码"写下了一段长长的记录。在最后，他写道：

```
// 1859.12.18 - 从查尔斯镇返回途中
// 核心判断：布朗事件是南北关系的不可逆转折点
// 从这一刻起，双方的恐惧和敌意只会增加，不会减少
// 战争概率评估：从60%上升到85%

// 行动计划更新：
// Phase 1: 武装力量建设 (deadline: 1860年秋)
//   - 庄园自卫队: 200人
//   - 武器装备: 200杆步枪 + 弹药储备
//   - 训练体系: 现代步兵战术 (散兵线 + 壕沟战)
// Phase 2: 资产转移 (deadline: 1861年春)
//   - 将南方固定资产的50%转化为可移动资产
//   - 建立海外贸易渠道 (英国/法国)
//   - 黄金储备: 目标$50,000
// Phase 3: 庄园防御工事 (deadline: 1861年春)
//   - 壕沟 + 木栅栏 + 了望塔
//   - 粮食储备: 6个月
//   - 医疗设施: 简易野战医院

// 关键人脉清单:
// R.E.Lee - 已建立联系，未来南军核心将领
// J.Stuart - 已加入铁路项目，未来骑兵天才
// S.Humphries - 政治盟友，但立场可能右转
// Old Tom - 黑人社区核心，解放计划的关键
```

他合上日记本，看向窗外。火车正穿越南卡罗来纳的松林地带，高大笔直的火松像一排排沉默的卫兵，在冬日的灰色天空下列队而立。

在这片美丽的、被诅咒的、注定要被鲜血浸透的土地上，阿欢闭上了眼睛。

不是入睡，而是在思考。

思考如何在一场不可避免的风暴中，保护他所能保护的一切。
