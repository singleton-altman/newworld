# 穿越之我在美国南部种棉花

> 一个华夏大龄程序员，穿越到1856年美国南部，用现代知识改写历史的故事。

---

## 简介

赵欢，38岁，华夏国互联网大厂程序员，被裁员后决定移民阿梅利卡"躺平"。飞机遭遇异常气流，他意外坠入1856年的美国南部佐治亚州——距离南北战争爆发还有五年。

没有公路，没有电，没有信号。只有一部电量57%的手机，和一颗程序员的脑袋。

从橡树庄园的棉花田开始，他用轮作制改良农业、建立纺织工坊、推动南方工业革命、组建现代军队——最终带领南方打败华盛顿，统一全国。

这不是一个简单的穿越爽文。这是一个关于**自由**的故事：奴隶的自由、穷人的自由、以及一个穿越者从"逃离内卷"到"建造文明"的灵魂自由。

## 核心人物

- **阿欢（赵欢）** — 穿越者，程序员思维，从庄园合伙人到国家缔造者
- **伊丽莎白·沃克** — 橡树庄园女主人，阿欢的妻子
- **老汤姆** — 黑奴领袖，南方第一位黑人军官
- **杰布·斯图尔特** — 骑兵天才，阿欢的战友
- **比利·麦考伊** — 机械天才，南方工业化核心工程师
- **李将军** — 南军统帅，传统军事荣耀的代表
- **格兰特** — 北军总司令，阿欢的终极对手
- **林肯** — 北方总统，理想与现实的矛盾体
- **铃木源吾** — 日本浪人，阿欢的贴身护卫
